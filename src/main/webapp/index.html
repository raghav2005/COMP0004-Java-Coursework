Hello World!

This is a standard html web page with no dynamic content. It can be accessed via:

  • http://localhost:8080, the default index.html page for the website.
  • or, http://localhost:8080/index.html if the index.html document name is included in the URL.

These links invoke the Java Servlet and Java Server Page (JSP) examples:

  • Java Servlet. The document name hello.html is mapped to the servlet called HelloWorldServlet, which dynamically creates the actual html document.
  • Java Servlet and JSP. The document name helloworldservletandjsp.html is mapped to the servlet HelloWorldServletAndJSP, which uses the helloworld.jsp file to dynamically display a message.
